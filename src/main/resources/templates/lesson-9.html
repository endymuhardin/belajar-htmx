File Upload

Menangani upload file secara asynchronous dengan progress bar real-time, tanpa perlu reload halaman penuh.

üìÇ Mekanisme Upload

Checklist Wajib:

  • ‚úì
    POST Method Gunakan hx-post, bukan GET.
  • ‚úì
    Multipart Encoding Wajib set hx-encoding="multipart/form-data".
  • ‚úì
    Progress Indicator Gunakan hx-indicator untuk feedback visual.

Bagaimana HTMX Bekerja:

  1. Intercept: HTMX menangkap event submit form.
  2. Package: Membungkus file dalam FormData object.
  3. Send: Mengirim AJAX request dengan progress events.
  4. Swap: Update area target dengan respon server.

Frontend (Form)

upload-form.html 
<form hx-post="/upload"
      hx-encoding="multipart/form-data"
      hx-target="#file-list"
      hx-indicator="#progress">
      
    <input type="file" name="file">
    <button>Upload</button>

    <!-- Progress Bar (Hidden by default) -->
    <progress id="progress" class="htmx-indicator" max="100"></progress>
</form>

Backend Controller

DocumentController.java 
@PostMapping("/upload")
public String upload(@RequestParam("file") MultipartFile file,
                     Model model) {
                     
    // 1. Simpan file ke disk
    storageService.store(file);
    
    // 2. Simpan metadata ke DB
    repo.save(new Document(file));
    
    // 3. Return updated list fragment
    return "fragments/file-list";
}
