Forms & Validation

Menangani input pengguna secara elegan: mengirim form dengan hx-post dan menampilkan validasi sisi server tanpa refresh halaman.

üìö Konsep Utama

HTMX mengubah cara kita menangani form. Alih-alih submit full page, kita kirim data secara background dan menerima potongan HTML yang berisi status sukses atau form yang diperbarui dengan pesan error.

Atribut Form

  • hx-post="/url" Kirim data form via POST request.
  • hx-target="#id" Area update (misal: div pembungkus form).
  • @Valid Anotasi Spring untuk validasi DTO otomatis.

Alur Validasi

  1. Form di-submit ke server
  2. Spring memvalidasi input (Bean Validation)
  3. Jika Error: Return fragmen form dengan pesan error
  4. Jika Sukses: Return pesan sukses / fragmen lain

üéÆ Demo Interaktif

Cobalah Skenario Ini:

  • Submit form kosong (lihat error global).
  • Isi nama < 3 karakter.
  • Isi email dengan format ngawur (tanpa @).
  • Isi semua dengan benar untuk melihat pesan sukses.

Frontend (HTML)

contact-form.html 
<form hx-post="/contact"
      hx-target="#form-container">
      
    <div>
        <label>Email</label>
        <input type="email" name="email" 
               th:classappend="${errors.has('email')} ? 'error'" />
        <span th:if="${errors.has('email')}" 
              th:text="${errors.get('email')}"></span>
    </div>

    <button type="submit">Kirim</button>
</form>

Backend (Java)

HomeController.java 
@PostMapping("/contact")
public String submit(@Valid ContactForm form, 
                     BindingResult result) {
                     
    if (result.hasErrors()) {
        // Validation failed: return form again with errors
        return "fragments/contact-form";
    }
    
    // Success: return success message
    return "fragments/contact-success";
}

Poin Penting

  • ‚óè Security: Validasi tetap dilakukan di server (Java), client (HTMX) hanya menampilkan hasilnya.
  • ‚óè User Experience: State form terjaga saat terjadi error (user tidak perlu mengetik ulang).
  • ‚óè No JavaScript: Validasi canggih tanpa menulis satu baris pun JS validasi di frontend.
