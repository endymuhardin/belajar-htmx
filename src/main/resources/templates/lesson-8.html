Infinite Scroll

Teknik memuat konten secara terus-menerus saat pengguna menggulir halaman, menciptakan pengalaman "tanpa akhir" yang mulus dengan hx-trigger="revealed".

üîÑ Flow Mekanisme

1

User Scrolls

Pengguna menggulir halaman ke bawah mendekati akhir daftar konten.

2

Trigger Revealed

Elemen khusus di akhir daftar (sentinel) masuk ke viewport ("revealed").

hx-trigger="revealed"
3

Fetch Next Page

HTMX mengirim request GET untuk halaman data berikutnya.

4

Swap & Append

Konten baru disisipkan setelah elemen terakhir. Sentinel dipindah ke paling bawah.

hx-swap="afterend"

The Trigger Sentinel

fragments/notes-page.html 
<!-- Data List -->
<div th:each="note : ${notes}">...</div>

<!-- Sentinel Element -->
<div th:if="${hasMore}"
     hx-get="/notes/page?page=${nextPage}"
     hx-trigger="revealed"
     hx-swap="afterend"
     hx-indicator="#spinner">
     
     <!-- Loading Spinner -->
     <div id="spinner" class="htmx-indicator">
        Loading...
     </div>
</div>

Backend Controller

NoteController.java 
@GetMapping("/notes/page")
public String page(@RequestParam int page, Model model) {
    Page<Note> data = repo.findAll(PageRequest.of(page, 10));
    
    model.addAttribute("notes", data.getContent());
    model.addAttribute("hasMore", data.hasNext());
    model.addAttribute("nextPage", page + 1);
    
    // Return partial fragment only!
    return "fragments/notes-page"; 
}
