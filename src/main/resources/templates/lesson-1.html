HTMX Basics

Memahami konsep fundamental hypermedia: bagaimana menggunakan hx-get dan hx-target untuk interaksi server.

üìö Konsep Utama

Ide Utama: HTMX memungkinkan elemen HTML (bukan hanya <a> atau <form>) untuk melakukan HTTP request dan mengganti bagian tertentu dari halaman dengan respons dari server.

Atribut HTMX Dasar

  • hx-get="/url" Mengirim GET request ke endpoint tertentu.
  • hx-target="#id" Menentukan elemen mana yang akan di-update dengan respons.
  • hx-swap="innerHTML" Metode penggantian konten (default: isi dalam target).

Alur Kerja

  1. User klik button dengan hx-get
  2. HTMX melakukan AJAX request
  3. Server merender fragmen HTML
  4. HTMX menyisipkan HTML tersebut ke target
