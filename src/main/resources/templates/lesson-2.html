Triggers & Swap Strategies

Menguasai kontrol penuh atas interaksi: kapan request dikirim (hx-trigger) dan bagaimana konten diperbarui (hx-swap).

Fleksibilitas: HTMX tidak hanya terbatas pada event 'click'. Kita bisa memicu request berdasarkan berbagai event DOM, waktu interval, atau bahkan event custom.

hx-trigger Options

  • load Trigger otomatis saat elemen dimuat (lazy loading).
  • click Default untuk button & link (klik user).
  • every 5s Polling berkala setiap interval waktu.

hx-swap Options

  • innerHTML Ganti isi target (Default).
  • outerHTML Ganti seluruh elemen target termasuk tag-nya.
  • beforeend Tambahkan di baris terakhir (Append).
